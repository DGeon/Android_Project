# 사용방법

</div>

#### 회원가입 시 연봉을 입력하고, 업무시간을 입력하게 된다. 이 후 어플을 자동로그인으로 실행하게 되면, 퇴근기록으로 연결되고 입력을 하면 실시간 버스 정보로 버스를 조회 한다. 업무일지를 기록 할 수 있고 야근을 하게 된다면 기존에입력했던 업무시간외는 야근 수당으로 계산이 되고, 월별 야근 수당을 보여준다
